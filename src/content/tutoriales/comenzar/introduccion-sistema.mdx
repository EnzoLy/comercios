# Bienvenido al Sistema de Gestión Comercial

¡Bienvenido! Este tutorial te ayudará a entender las funcionalidades principales del sistema y cómo pueden ayudarte a administrar tu negocio de manera eficiente.

<Callout type="info" title="¿Qué es el Sistema de Gestión Comercial?">
Es una plataforma completa para administrar tu tienda o negocio. Incluye punto de venta (POS), gestión de inventario, control de empleados, reportes de ventas y mucho más.
</Callout>

## Principales Módulos del Sistema

El sistema está organizado en varios módulos especializados que trabajan juntos para ofrecerte una solución completa:

<InfoCard icon="ShoppingCart" title="Punto de Venta (POS)">
Registra ventas de forma rápida y sencilla. Escanea códigos de barras, aplica descuentos, acepta múltiples métodos de pago y genera tickets automáticamente.
</InfoCard>

<InfoCard icon="Package" title="Gestión de Productos">
Administra tu catálogo completo de productos. Configura precios, stock, categorías, proveedores y códigos de barras. Controla fechas de vencimiento para productos perecederos.
</InfoCard>

<InfoCard icon="Archive" title="Control de Inventario">
Mantén un registro detallado de todos los movimientos de stock. Visualiza alertas cuando los productos están por agotarse y realiza ajustes de inventario.
</InfoCard>

<InfoCard icon="Users" title="Administración de Empleados">
Gestiona tu equipo de trabajo. Asigna roles y permisos, controla turnos de trabajo y realiza cierres de caja por empleado.
</InfoCard>

<InfoCard icon="BarChart3" title="Reportes y Analíticas">
Toma decisiones informadas con reportes detallados de ventas, productos más vendidos, rendimiento por empleado y análisis de tendencias.
</InfoCard>

<InfoCard icon="Truck" title="Gestión de Proveedores">
Mantén un directorio completo de tus proveedores. Registra órdenes de compra, controla pagos y mantén historial de transacciones.
</InfoCard>

## Primeros Pasos

<StepList>
  <Step title="Explora el Panel de Control">
    El panel de control es tu punto de partida. Aquí encontrarás un resumen de las métricas más importantes: ventas del día, productos con bajo stock, turnos activos y más.
  </Step>

  <Step title="Configura tus Productos">
    Antes de realizar ventas, necesitas agregar tus productos al sistema. Ve a la sección "Productos" y comienza a crear tu catálogo. Puedes agregar productos uno por uno o importarlos masivamente desde Excel.
  </Step>

  <Step title="Registra tu Primera Venta">
    Una vez que tengas productos en el sistema, dirígete al módulo de "Caja" (POS) para realizar tu primera venta. Escanea productos, selecciona el método de pago y genera el ticket.
  </Step>

  <Step title="Revisa tus Reportes">
    Al final del día, consulta tus reportes de ventas para ver el rendimiento de tu negocio. Analiza qué productos se vendieron más y cuánto efectivo ingresó.
  </Step>
</StepList>

## Sistema de Roles y Permisos

<Callout type="warning" title="Importante: Roles en el Sistema">
El sistema maneja diferentes roles de usuario, cada uno con permisos específicos:

- **Propietario/Admin**: Acceso completo a todas las funcionalidades
- **Gerente**: Puede gestionar empleados, ver reportes y realizar ventas
- **Encargado de Stock**: Gestiona inventario, productos y realiza ventas
- **Cajero**: Solo puede realizar ventas en el punto de venta
</Callout>

Los tutoriales disponibles para ti dependerán de tu rol en el sistema. Algunos tutoriales avanzados solo estarán disponibles para administradores.

## Modo Oscuro

El sistema incluye modo oscuro para trabajar cómodamente en cualquier condición de iluminación. Puedes activarlo/desactivarlo usando el botón en la esquina superior derecha.

## Soporte Offline

<Callout type="success" title="Trabaja sin conexión">
El punto de venta funciona incluso sin conexión a internet. Las ventas se guardan localmente y se sincronizan automáticamente cuando recuperas la conexión.
</Callout>
